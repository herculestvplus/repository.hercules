# Repositório Hercules TV para Kodi

Bem-vindo ao repositório Hercules TV! Este guia vai ajudá-lo a instalar o repositório no seu Kodi.

## Como Instalar

### Método 1: Instalação Direta via URL

1. Abra o **Kodi**
2. Clique no ícone de **Configurações** (engrenagem) no menu principal
3. Selecione **Gerenciador de ficheiros** (ou **File Manager**)
4. Clique em **Adicionar fonte** (ou **Add source**)
5. Clique em **<Nenhum>** (ou **<None>**)
6. Digite o seguinte URL: `https://tinyurl.com/herculestv`
7. Clique em **OK**
8. No campo de nome, digite: **Hercules TV**
9. Clique em **OK** novamente

### Método 2: Instalar o Repositório

1. No menu principal do Kodi, vá para **Add-ons**
2. Clique no ícone da **caixa aberta** no canto superior esquerdo
3. Selecione **Instalar de um ficheiro zip** (ou **Install from zip file**)
4. Selecione **Hercules TV** (a fonte que você acabou de adicionar)
5. Selecione o ficheiro `.zip` do repositório
6. Aguarde a notificação de que o repositório foi instalado

### Método 3: Instalar Add-ons do Repositório

1. No menu de Add-ons, clique novamente no ícone da **caixa aberta**
2. Selecione **Instalar do repositório** (ou **Install from repository**)
3. Selecione **Repositório Hercules TV**
4. Navegue pelas categorias e escolha os add-ons que deseja instalar
5. Clique em **Instalar**

## Resolução de Problemas

### "Instalação de fontes desconhecidas está desativada"

Se receber esta mensagem:

1. Vá para **Configurações** → **Sistema** → **Add-ons**
2. Ative a opção **Fontes desconhecidas** (ou **Unknown sources**)
3. Confirme o aviso de segurança
4. Volte e tente instalar novamente

### "Não foi possível conectar à fonte"

- Verifique sua conexão com a internet
- Certifique-se de que digitou o URL corretamente: `https://tinyurl.com/herculestv`
- Tente novamente após alguns minutos

## Suporte

Se encontrar algum problema ou tiver dúvidas sobre os add-ons disponíveis, entre em contato através dos canais de suporte do Hercules TV.
